# Curiosidades sobre Linux (Português)

Uma é exibida ao acaso no modo Linux. Cada linha `- ` é uma curiosidade — adicione/edite à vontade.

- O Linux foi iniciado em 1991 por Linus Torvalds, então um estudante de 21 anos em Helsinque, como um hobby.
- O Tux, mascote do Linux, é um pinguim — supostamente porque Linus gosta de pinguins (e teria sido beliscado por um).
- O Tux foi desenhado por Larry Ewing em 1996 usando o GIMP.
- O Android, o sistema operacional mais usado do mundo, é construído sobre o kernel do Linux.
- Todos os 500 supercomputadores mais rápidos do mundo rodam Linux.
- O nome "Linux" mistura "Linus" e "Unix".
- Linus queria chamá-lo de "Freax"; o nome "Linux" veio de uma pasta em um servidor FTP.
- "GNU" é um acrônimo recursivo para "GNU's Not Unix" (GNU Não é Unix).
- O kernel do Linux cresceu de cerca de 10.000 linhas de código em 1991 para dezenas de milhões hoje.
- Os notebooks a bordo da Estação Espacial Internacional rodam Linux.
- O helicóptero Ingenuity da NASA, em Marte, rodava Linux.
- "sudo" significa "superuser do" (fazer como superusuário).
- "Ubuntu" é uma palavra Nguni que significa, aproximadamente, "humanidade para com os outros".
- O Linux é livre e de código aberto, lançado sob a licença GPL.
- Existem centenas de distribuições Linux — Debian, Ubuntu, Fedora, Arch e muitas outras.
- O Steam Deck roda o SteamOS, que é baseado no Arch Linux.
- O clássico comando "fortune" exibe uma frase aleatória — a inspiração para este pequeno terminal.
- A maioria dos servidores que mantêm a internet no ar roda Linux.
