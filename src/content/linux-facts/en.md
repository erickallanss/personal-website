# Linux curiosities (English)

One is shown at random in Linux mode. Each `- ` line is one fact — add/edit freely.

- Linux was started in 1991 by Linus Torvalds, then a 21-year-old student in Helsinki, as a hobby.
- Tux, the Linux mascot, is a penguin — reportedly because Linus is fond of penguins (and was once nibbled by one).
- Tux was drawn by Larry Ewing in 1996 using GIMP.
- Android, the world's most-used operating system, is built on the Linux kernel.
- All 500 of the world's fastest supercomputers run Linux.
- The name "Linux" blends "Linus" and "Unix."
- Linus first wanted to call it "Freax"; the name "Linux" came from a folder on an FTP server.
- "GNU" is a recursive acronym for "GNU's Not Unix."
- The Linux kernel grew from about 10,000 lines of code in 1991 to tens of millions today.
- The laptops aboard the International Space Station run Linux.
- NASA's Ingenuity Mars helicopter ran on Linux.
- "sudo" stands for "superuser do."
- "Ubuntu" is an Nguni word meaning roughly "humanity toward others."
- Linux is free and open source, released under the GPL.
- There are hundreds of Linux distributions — Debian, Ubuntu, Fedora, Arch, and many more.
- The Steam Deck runs SteamOS, which is based on Arch Linux.
- The classic "fortune" command prints a random saying — the inspiration for this little terminal.
- Most of the servers that run the internet run Linux.
