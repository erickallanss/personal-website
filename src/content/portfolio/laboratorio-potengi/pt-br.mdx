## Visão geral

Um sistema de gestão clínica feito sob medida para o **Laboratório Potengi** — uma ferramenta
focada, e não um produto genérico.

## O que faz

- Marcação de consultas
- Prontuário eletrônico
- Fila de atendimentos em tempo real

## Impacto

Feito sob medida para o fluxo de trabalho real do laboratório, ajudou a equipe a organizar e
acelerar o dia a dia.

## Stack

Django + Django REST Framework no backend, frontend em React + Tailwind CSS, implantado na AWS
(Amplify, EC2, S3).
