## Overview

A clinical management system built specifically for **Laboratório Potengi** — a focused,
made-to-measure tool rather than a one-size-fits-all product.

## What it does

- Appointment scheduling
- Electronic medical records (prontuário)
- A live attendance queue

## Impact

Built bespoke around the lab's real workflow, it helped the team organize and speed up their
day-to-day operations.

## Stack

Django + Django REST Framework on the backend, a React + Tailwind CSS frontend, deployed on AWS
(Amplify, EC2, S3).
