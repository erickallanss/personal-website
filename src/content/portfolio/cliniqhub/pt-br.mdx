## Visão geral

Um **SaaS** completo de gestão para clínicas — reunindo a operação do dia a dia, a previsibilidade financeira e os fluxos clínicos em uma única plataforma.

## O que faz

- Gestão do dia a dia da clínica
- Previsibilidade financeira e relatórios
- Fluxos clínicos

## Feito para escalar

Um SaaS multi-tenant projetado para se manter performático conforme a base de clientes cresce.

## Stack

Django + Django REST Framework no backend, frontend em React + Tailwind CSS e uma ampla
infraestrutura AWS (EC2, S3, SES, SNS, SQS, RDS, CloudWatch, Amplify, ECR) provisionada com Terraform.

> O produto é um SaaS privado — o link leva à landing page.
