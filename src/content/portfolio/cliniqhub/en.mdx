## Overview

A complete clinic-management **SaaS** — bringing day-to-day operations, financial predictability, and clinical workflows together in one platform.

## What it does

- Day-to-day clinic management
- Financial predictability and reporting
- Clinical workflows

## Built to scale

A multi-tenant SaaS designed to stay performant as the customer base grows.

## Stack

Django + Django REST Framework on the backend, a React + Tailwind CSS frontend, and a broad AWS
footprint (EC2, S3, SES, SNS, SQS, RDS, CloudWatch, Amplify, ECR) provisioned with Terraform.
